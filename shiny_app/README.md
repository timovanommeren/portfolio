# Counting the Returns - MDMA Regulation Through a Financial Lens

This repository contains the R files used for  calculating the estimate presented in the policy report *Counting the Returns: MDMA Regulation Through a Financial Lens*. An estimate of the total revenue of an hypothetical MDMA state-run monopoly was calculated using the following formula:

**Demand = Number of Consumers x Frequency of Consumption x Intensity of Consumption.**

**Annual revenue = Demand × Market capture × (Price − Variable cost)**.

## Shiny APP

To re-run the analysis presented in policy report *Counting the Returns: MDMA Regulation Through a Financial Lens* using different estimates, visit the [Estimating MDMA Monopoly Revenues Shiny APP](https://timovanommeren.github.io/MDMA_financial_revenues_quickscan/)
